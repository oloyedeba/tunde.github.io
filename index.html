🚀 How to Deploy Your App

GitHub Pages

  1. Create a new GitHub repo
  2. Upload your project files
  3. Go to Settings → Pages
  4. Set source to main branch
  5. Live at: username.github.io/repo

Netlify (easiest)

  1. Go to netlify.com → sign up free
  2. Drag & drop your project folder
  3. Get an instant live URL
  4. Add custom domain (optional)
  5. Auto-deploys on every update
⚠️ Security: Share API keys privately via Google Classroom. Never put keys in public repos — teach students to use environment variables for real deployments.
