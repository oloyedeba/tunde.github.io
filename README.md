## 🚀 What It Does

CollegeCoach AI is an agentic AI assistant that helps high school students with every part of the college application process:

- 💡 **Brainstorm** — Discover your unique stories and what makes you stand out
- ✏️ **Essay Feedback** — Get specific, actionable feedback on your drafts
- 📝 **Prompt Help** — Understand any essay prompt and find the best angle
- 🏫 **School Match** — Tailor your supplemental essays to specific colleges

---

## 🛠 How It's Built

| Technology | Purpose |
|---|---|
| HTML / CSS / JavaScript | Frontend — no frameworks needed |
| Claude API (claude-sonnet-4) | AI responses via Anthropic |
| GitHub Pages / Netlify | Free hosting & deployment |

**No backend. No database. No build tools.** Just one HTML file.

---

## 🧠 AI Concepts Demonstrated

- **System prompts** — Shapes the AI's persona, tone, and rules
- **Multi-turn conversations** — Maintains context across a full session
- **Prompt engineering** — Iteratively improved to give better coaching
- **AI safety & guardrails** — Prevents the AI from writing essays *for* students
- **Agentic behavior** — AI that guides users toward a goal step by step

## 🌐 Deploy Your Own

### Netlify (recommended — 2 minutes)
1. Go to [netlify.com](https://netlify.com) and sign up free
2. Drag the project folder onto the Netlify dashboard
3. Done — you have a live URL instantly

### GitHub Pages
1. Create a new GitHub repo
2. Upload `index.html` and `.nojekyll`
3. Go to **Settings → Pages → Source: main branch**
4. Your app is live at `username.github.io/repo-name`

## ⚠️ API Key Security

This app asks for a Claude API key on launch. The key is:
- ✅ Stored only in browser memory (gone when you close the tab)
- ✅ Never sent anywhere except the Anthropic API
- ❌ Never hardcode your API key in any file you push to GitHub

For classroom use, the teacher shares the key privately via Google Classroom.
For personal use, get a free key at [console.anthropic.com](https://console.anthropic.com).
